# 福井県観光ロケーション・トレンドレポート

## 概要

このアプリケーションは、福井県内のオンラインマップアプリケーション及びWeb検索ツールでのインプレッション回数を集計し、トレンドをグラフで可視化したものです。
[アプリを開く](https://code4fukui.github.io/fukui-kanko-trend-report/)

### 主な機能

- **地域別・全体集計**: 福井県全体、または特定地域のデータを表示
- **時系列分析**: 日次・週次・月次でデータを集計して表示
- **2期間比較**: 異なる期間のデータを並べて比較
- **多様な指標**: 地図検索、Web検索、ルート検索、通話、ウェブサイトクリック、レビュー投稿数、星別レビュー数、平均評点
- **CSVエクスポート**: 表示中のデータをCSV形式でダウンロード

## 対象地域

福井県内の以下13地域のデータを収集しています：

- あわら市
- 池田町
- おおい町
- 永平寺町
- 越前市
- 勝山市
- 南越前町
- 高浜町
- 敦賀市
- 若狭町
- 小浜市
- 美浜町
- 大野市

## 使い方

1. **エリア選択**: 「全域」または特定の地域を選択
2. **時間単位選択**: 日別・週別・月別から表示単位を選択
3. **期間選択**: カレンダーから分析したい期間を選択
4. **2期間比較**: チェックボックスをONにして比較期間を設定
5. **データ確認**: グラフで「回数推移」と「レビュー推移」を確認
6. **CSVダウンロード**: 必要に応じてデータをCSV形式でエクスポート

## データ形式

### CSVファイル構造

各CSVファイルには以下のカラムが含まれます：

| カラム名               | 説明                   |
| ---------------------- | ---------------------- |
| `日付`                 | 日付（YYYY-MM-DD）     |
| `地図検索`             | 地図検索回数           |
| `Web検索`              | Web検索回数            |
| `ルート検索`           | ルート検索回数         |
| `通話`                 | 通話ボタンクリック数   |
| `ウェブサイトクリック` | ウェブサイトクリック数 |
| `レビュー投稿数`       | レビュー投稿数         |
| `星5_レビュー数`       | 星5のレビュー数        |
| `星4_レビュー数`       | 星4のレビュー数        |
| `星3_レビュー数`       | 星3のレビュー数        |
| `星2_レビュー数`       | 星2のレビュー数        |
| `星1_レビュー数`       | 星1のレビュー数        |
| `平均評点`             | 平均評点               |

## 開発者向け資料

### インストール手順

1. リポジトリをクローン

```bash
git clone https://github.com/code4fukui/fukui-kanko-trend-report.git
cd fukui-kanko-trend-report
```

2. サブモジュールを含めて依存関係をインストール

```bash
git submodule update --init --recursive
pnpm install
```

**サブモジュールの更新（データを最新にする場合）**

```bash
git submodule update --remote --recursive
```

3. 開発サーバーを起動

```bash
pnpm dev
```

4. ブラウザで `http://localhost:5173` にアクセス
